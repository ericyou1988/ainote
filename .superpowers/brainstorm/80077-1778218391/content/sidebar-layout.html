方案 C：标签页工作区

顶部导航栏含笔记/分析/对话/设置标签页。主区域始终为单栏，通过标签页切换功能模块。类似浏览器标签页的概念，适合习惯浏览器操作的用户。移动端用底部标签栏。

核心区别

  • A = 侧边栏 + 全屏内容（最省空间，列表被隐藏）
  • B = 两栏同时可见（列表和内容并存，适合 PC 大空间）
  • C = 单栏 + 顶部标签切换（最接近现状，改动最小）
